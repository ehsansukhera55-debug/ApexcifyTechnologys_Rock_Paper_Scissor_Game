# 🪨📄 Rock Paper Scissors Pro

A modern, dark-themed Graphical User Interface (GUI) version of the classic Rock Paper Scissors game, built using Python and **CustomTkinter**.

##  Features
* **Modern UI:** Sleek dark mode interface with rounded buttons.
* **Real-time Scoring:** Tracks your wins vs. the computer.
* **Visual Feedback:** Dynamic text colors for wins, losses, and ties.
* **Responsive Logic:** Interactive buttons instead of console inputs.
